# Karthikraj Nadar - Professional Portfolio

Interactive Computational Design and Architecture Portfolio showcasing projects across BIM Automation, Generative Algorithms, and Architectural Design & Detailing.

This dynamic React web application acts as an interactive digital CV, collaboratively built leveraging **Google AI Studio**. It features dual viewing modes emphasizing two distinct professional disciplines:
- **VDC & Computational Core (Neon Terminal Theme):** Highlights algorithmic solvers, Grasshopper/Rhino/Revit API scripting, data serialization, and 5D/4D workflow automations.
- **Architecture & Ecosystems (Clean Spatial Theme):** Showcases pure architectural design, ecological planning, bespoke woodworking, modular prefabrication, and sustainable space-making.

## 🛠️ Stack & Technologies

This portfolio is built from scratch as a modern Single Page Application (SPA), utilizing the latest web development tooling to ensure a premium, highly responsive user experience. 

- **Frontend:** React 19 with strong TypeScript typing
- **Styling:** Tailwind CSS 4 for utility-first layout processing
- **Animations:** `motion/react` for complex spring transitions, smooth reveals, and scroll-spy interactions
- **Icons:** Lucide React
- **Build Engine:** Vite 6
- **Server:** Built on a full-stack Node.js + Express setup for routing, mail bridging, and deploying static files smoothly on customized hosting pipelines.

## ✨ Key Technical Features

- **Dual-Identity Architecture in React:** Seamless, fully animated transition logic to instantly swap the website between a dark-mode "developer terminal" style and a pristine, high-contrast typography "architecture layout" without full-page reloads.
- **Animated Layout Grids:** Highly polished project showcase cards with interactive media expansion, dynamic SVGs, and sliding timeline reveals built with `motion`.
- **Responsive Staggered Reflow:** Graceful resizing logic shifting rigid multi-column layouts into focused mobile feeds with 100% responsiveness.
- **Backend Communication Bridge:** Secured Express backend logic to process form submissions and routing.

## ⚙️ Running Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/karthikraj-v-nadar/portfolio.git
   cd portfolio
   ```

2. **Install dependencies:**
   Make sure you have Node > 18 installed, then run:
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
   *The server will start at `http://localhost:3000` (or another port based on your local environment).*

4. **Production Build:**
   ```bash
   npm run build
   npm start
   ```
